\# descriptiveStats



An R package providing descriptive statistics helper functions.



\## Functions



\- `calc\_mean()` - arithmetic mean

\- `calc\_median()` - median

\- `calc\_mode()` - mode

\- `calc\_q1()` - first quartile

\- `calc\_q3()` - third quartile

\- `calc\_iqr()` - interquartile range



\## Notes



The package includes:

\- input validation

\- handling of NA values

\- empty vector handling

\- simple unit tests using `testthat`
