\# Question 1 – Descriptive Statistics R Package



This folder contains the `descriptiveStats` R package developed for the coding assessment.



The package implements several descriptive statistics functions:



\- calc\_mean()

\- calc\_median()

\- calc\_mode()

\- calc\_q1()

\- calc\_q3()

\- calc\_iqr()



The package follows standard R package structure with documentation and test cases.
